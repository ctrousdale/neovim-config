# Neovim (nvim/)

## Overview
Neovim configuration entrypoint and plugin bootstrap.

## Where To Look
- Entry: `nvim/init.lua`
- Plugin bootstrap: `nvim/lua/plugins/init.lua`
- Plugin package registry: `nvim/lua/plugins/init.lua`
- Formatter map: `nvim/lua/plugins/conform.lua`
- Linter map: `nvim/lua/plugins/lint.lua`

## Guardrails
- `nvim/nvim-pack-lock.json` is generated by `vim.pack` (do not hand-edit).

## Notes
Plugin-specific configuration lives under `nvim/lua/plugins/`.
